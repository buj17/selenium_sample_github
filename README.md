## Инструкция по запуску
Для запуска понадобится:
- git
- uv. Инструкция по его установке [здесь](https://docs.astral.sh/uv/getting-started/installation/)

### Шаги для запуска:
#### Склонировать репозиторий
```bash
git clone https://github.com/buj17/selenium_sample_github
```

#### Перейти в директорию с проектом
```bash
cd selenium_sample_github
```

#### Установить зависимости
```bash
uv sync
```

#### Запустить пример решения задачи с предыдущей темы
```bash
uv run main.py
```
